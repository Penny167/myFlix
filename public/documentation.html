myFlixapi

Overview

The myFlixapi is a REST API that supports the "myFlix" movie web application, by providing access to the server-side database. This contains information about movies, including: title, genre, director and more.

Table of API endpoints

The following table lists all available endpoints including a high level description of the data available, URL, http method, format required for requests and format of responses. Request parameters within the URL are indicated with square brackets.

Description URL HTTP method Request body data format Response body data format
Return a list of all movies /movies GET No request body required JSON object with data on all movies
Return data about a single movie by title /movies/[title] GET No request body required JSON object with data about movie requested: { title: description: genre: director: image url: }
Return the movie genre description by title /movies/[title]/[genre] GET No request body required A text message returning the genre of the movie selected
Return data about a director by name /directors/[name] GET No request body required JSON object with data about director requested: { name: bio: birth year: death year: }
Register a new user /users POST A JSON object with data about the user to be registered: { username: password: favourites: [ ] } A text message confirming that registration has been successful
Update a users username /users/[username]/[newUsername] PUT No request body required A text message confirming that the username has been updated
Add a movie to users list of favourites (this updates the list of favourites associated to that user) /users/[username]/[favourites]/[newFavourite] PUT No request body required A text message confirming that the movie has been added to favourites
Remove a movie from users list of favourites /users/[username]/[favourites] DELETE No request body required A text message confirming that the movie has been removed from favourites
Deregister a user /users/[username] DELETE No request body required A text message confirming that the user has been deregistered
